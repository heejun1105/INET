엑셀 파일 업로드 도움말

업로드 시 다음 사항을 확인해주세요:

  • 파일은 .xls 또는 .xlsx 형식이어야 합니다.
  • 첫 번째 행은 헤더로 간주됩니다.
  • 필수 입력 항목: 장비 타입(3번째 열), 설치장소(10번째 열)
  • 관리번호는 '카테고리-번호' 또는 '카테고리-연도-번호' 형식이어야 합니다.
  • 도입일자는 다양한 형식으로 입력 가능합니다:
    • YYYY-MM-DD: 2025-03-15
    • YYYY-MM: 2025-03 (일자는 1일로 자동 설정)
    • YYYY: 2025 (월/일은 1월 1일로 자동 설정)
    • YY-MM: 25-03 (현재 세기에 맞춰 연도 해석)
    • YY: 25 (현재 세기에 맞춰 연도 해석, 월/일은 1월 1일로 자동 설정)
    • 구분자로 ., /, - 또는 '년', '월', '일' 사용 가능
  • IP 주소는 자유 형식으로 입력 가능합니다.
