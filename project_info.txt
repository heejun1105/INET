INET 프로젝트 기술 스택 및 기능 설명
====================================

1. 기술 스택
-----------
1.1 백엔드
- Java 21 (JavaLanguageVersion.of(21))
- Spring Boot 3.2.3
- Spring Data JPA
- Spring Security
- MySQL 8 (데이터베이스)
- Gradle (빌드 도구)

1.2 프론트엔드
- Thymeleaf (템플릿 엔진)
- HTML/CSS
- JavaScript

1.3 추가 라이브러리
- Lombok
- Apache POI 5.2.3 (엑셀 처리)
- Cache2k 2.6.1.Final (캐싱)
- MySQL Connector/J

2. 주요 기능
-----------
2.1 장비 관리 시스템
- 장비 등록/수정/삭제
- 장비 목록 조회 (페이징 처리)
- 장비 필터링 (학교/교실/장비유형별)
- 엑셀 다운로드 기능

2.2 학교 관리
- 학교 정보 관리
- 학교별 장비 현황 조회
- 학교 구조도 시각화

2.3 교실 관리
- 교실 정보 관리
- 교실별 장비 배치 현황
- 교실 레이아웃 관리

2.4 운영자(담당자) 관리
- 운영자 정보 관리
- 학교별 운영자 배정
- 장비별 담당자 지정

2.5 검색 및 필터링
- 학교별 검색
- 교실별 검색
- 장비 유형별 검색
- 복합 조건 검색 (학교 + 교실 + 장비유형)

2.6 데이터 시각화
- 학교 구조도
- 교실 배치도
- 장비 배치 현황

2.7 보안 기능
- Spring Security 기반 인증/인가
- 사용자 권한 관리

2.8 데이터 내보내기
- 엑셀 형식의 장비 목록 다운로드
- 필터링된 데이터 내보내기

3. 주요 엔티티
------------
3.1 Device (장비)
- 관리번호, 유형, 제조사, 모델명
- 구매일자, IP주소, 용도
- 세트분류, 불용여부, 비고

3.2 School (학교)
- 학교명, 주소
- 학교 구조도 정보

3.3 Classroom (교실)
- 교실명, 건물, 층
- 위치 정보 (x, y 좌표)
- 크기 정보 (width, height)

3.4 Operator (운영자)
- 이름, 직위
- 담당 학교 정보
